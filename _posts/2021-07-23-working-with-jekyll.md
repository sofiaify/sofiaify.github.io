Jekyll is a static site generator that takes your content written in Markdown, runs it through templates, and produces a fast, secure website ready to host anywhere.

## Why static?

Static sites are fast, cheap to host, and secure by default. There is no database to hack and no server to keep patched.

> Jekyll does what you tell it to do — no more, no less.

## Getting started

Install Jekyll, create your content in Markdown, and let the build process handle the rest.

- Write content in Markdown
- Reuse layouts and includes
- Deploy to GitHub Pages for free
